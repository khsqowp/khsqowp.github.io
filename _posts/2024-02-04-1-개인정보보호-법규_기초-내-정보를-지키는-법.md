# 📜 1. 개인정보보호 법규_기초: 내 정보를 지키는 법

## 🎯 이 문서에서 배울 내용

이 문서를 끝까지 읽으면 다음을 할 수 있게 됩니다:

- ✅ 개인정보보호법이 왜 필요한지 이해하기
- ✅ 개인정보가 무엇인지 명확히 알기
- ✅ 기업이 지켜야 할 기본 규칙 알기
- ✅ 내 권리(정보 주체)가 무엇인지 알기

## 🤔 개인정보보호법이 왜 필요할까요?

### 💳 실생활 비유: 지갑을 맡길 때

```
상황: 가게에서 잠깐 짐을 맡겨야 해요

❌ 규칙이 없다면:
┌────────────────────────────────┐
│ 점원: 지갑 여기 놔두세요        │
│ - 마음대로 열어봄               │
│ - 사진 찍어서 친구에게 보냄     │
│ - 카드 번호 메모함              │
│ - 버리고 싶으면 버림             │
│                                │
│ 당신: "내 지갑인데...?" 😱      │
└────────────────────────────────┘

✅ 규칙이 있다면:
┌────────────────────────────────┐
│ 점원: 지갑 맡았습니다           │
│ - 절대 열어보지 않음            │
│ - 금고에 안전하게 보관          │
│ - 주인만 찾아갈 수 있음         │
│ - 일정 시간 후 반환             │
│                                │
│ 당신: "안심이 되네요!" 😊       │
└────────────────────────────────┘
```

**개인정보보호법 = 내 정보를 맡길 때의 규칙**

### 📊 실제 사고 사례

```
2014년 카드 3사 정보 유출 사건:
- 유출된 정보: 1억 400만 건 (!)
- 이름, 주민번호, 전화번호, 카드번호
- 영향: 전 국민의 2배

2018년 페이스북 정보 유출:
- 유출된 정보: 8,700만 명
- 정치 광고에 무단 사용됨
- 과징금: 50억 달러 (약 6조 원)
```

## 🆔 개인정보가 뭔가요?

### 📝 개인정보의 정의

```
개인정보 = 살아있는 사람에 대한 정보

예시:
✅ 이름 (홍길동)
✅ 주민등록번호 (901234-1234567)
✅ 전화번호 (010-1234-5678)
✅ 이메일 (hong@example.com)
✅ 주소 (서울시 강남구...)
✅ 사진, 동영상 (얼굴이 나오면)
✅ 자동차 번호 (12가 3456)
✅ IP 주소 (192.168.1.1)
✅ 쿠키 (웹사이트 방문 기록)
✅ 위치 정보 (GPS 기록)
```

### 🔒 특별히 더 보호해야 하는 정보 (민감정보)

```
민감정보:
🚫 사상·신념 (종교, 정치 성향)
🚫 노동조합 가입 여부
🚫 건강 정보 (병원 기록)
🚫 성생활 정보
🚫 유전 정보, 범죄 기록

→ 이것들은 절대 함부로 수집 금지!
→ 수집하려면 별도 동의 필요!
```

### ❓ 개인정보인지 헷갈리는 것들

```
Q1: 회사 이메일은 개인정보인가요?
   (예: hong.gildong@company.com)

A: 네! 이름이 포함되어 있으면 개인정보입니다.

Q2: "홍길동"만 있으면 개인정보인가요?

A: 네! 이름만으로도 특정 사람을 알아볼 수 있으면
   개인정보입니다.

Q3: IP 주소도 개인정보인가요?

A: 네! IP로 특정 사람을 추적할 수 있으면
   개인정보입니다.

Q4: 사망한 사람의 정보는?

A: 아니요. 개인정보보호법은 "살아있는 사람"만
   보호합니다.
```

✅ 목적 명확
✅ 다른 용도 사용 금지
```

```
동의를 받아야 하는 것:

필수 동의 (안 하면 서비스 이용 불가):
☑ 개인정보 수집·이용 동의
☑ 개인정보 제3자 제공 동의 (해당시)

선택 동의 (안 해도 서비스 이용 가능):
☐ 마케팅 정보 수신 동의
☐ 이벤트·혜택 정보 수신 동의

⚠️ 중요: 필수와 선택을 분리!
```

### 4️⃣ 규칙 4: 안전하게 보관하기

```
개인정보 보관 방법:

🔐 암호화:
- 주민번호, 비밀번호, 카드번호
- 해커가 훔쳐도 못 읽게!

🚪 접근 제한:
- 꼭 필요한 사람만 접근
- 접근 기록 남기기

💾 백업:
- 정기적으로 백업
- 백업도 암호화!

🗑️ 파기:
- 필요 없으면 바로 버리기
- 복구 불가능하게!
```

### 5️⃣ 규칙 5: 유출되면 즉시 알리기

```
개인정보 유출 시:

1단계: 즉시 파악 (24시간 내)
   - 언제, 어떻게, 얼마나?

2단계: 사용자에게 알림 (5일 내)
   - 이메일, 문자, 공지사항
   - "당신의 이름, 전화번호가 유출됐어요!"

3단계: 관계 기관에 신고 (5일 내)
   - 개인정보보호위원회
   - 한국인터넷진흥원

4단계: 피해 최소화
   - 비밀번호 변경 안내
   - 보안 조치 강화
```

```

4. 처리 정지권:
   "내 정보 사용하지 마세요!"

   "동의 취소할래요!"

→ 요청하면 10일 내 처리해야 함!

## 🌍 해외 법규: GDPR (유럽)

### 🇪🇺 GDPR이 뭔가요?

```
GDPR = 유럽 연합의 개인정보보호법

특징:
- 세계에서 가장 엄격
- 과징금이 엄청 큼
- 유럽인 대상이면 한국 기업도 적용!

예시:
한국 쇼핑몰이 유럽 사람에게
물건 팔면 → GDPR 적용!
```

### 🎁 특별한 권리: 잊힐 권리

```
잊힐 권리 (Right to be Forgotten):

상황:
"10년 전 실수로 인터넷에
 올린 내 사진... 지워주세요!"

기업:
"네, 삭제했습니다!"

→ 과거 정보를 삭제해달라고
   요청할 권리!
```

### 💸 엄청난 과징금

```
GDPR 과징금:

- 전 세계 매출의 4%

- 더 큰 금액으로!

예시:
연 매출 100억 원 회사:
→ 최대 4억 원

→ 최대 4천억 원!

   ☑ 카메라 (필요)

```
✅ 체크포인트:

   - 8자 이상
   - 영문+숫자+특수문자

4. 개인정보 유효기간 확인:
   "1년 안 쓰면 삭제됩니다" → 확인!
```

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 카페 회원가입 하는데 주민번호를 요구해요. 줘야 하나요?
**A:** 아니요!
```
주민번호 수집 가능한 곳:
✅ 은행, 증권사 (금융거래)
✅ 통신사 (본인 확인)
✅ 병원 (의료 서비스)

주민번호 수집 불가:
❌ 일반 쇼핑몰
❌ 카페
❌ 커뮤니티 사이트

→ 불법 요구면 거부하세요!
→ 개인정보보호위원회에 신고 가능
```

### Q2: 내 정보가 유출됐는지 어떻게 알아요?
**A:** 이렇게 확인하세요:
```
1. 회사 공지:
   이메일, 문자로 알림 받음

2. 이상 징후:
   - 모르는 곳에서 로그인 시도
   - 스팸 전화/문자 급증
   - 내가 안 산 물건 결제

3. 확인 사이트:
   - Have I Been Pwned (https://haveibeenpwned.com)
   - 내 이메일 입력 → 유출 여부 확인

4. 신용정보 조회:
   - 년 3회 무료 조회 가능
```

### Q3: 개인정보 유출되면 어떻게 해야 하나요?
**A:** 즉시 이렇게 하세요:
```
1단계: 비밀번호 변경 (즉시!)
   - 유출된 사이트
   - 같은 비밀번호 쓰는 다른 사이트

2단계: 2단계 인증 설정
   - 이메일, 문자 인증 추가

3단계: 금융 거래 확인
   - 카드 사용 내역
   - 계좌 이체 내역

4단계: 피해 신고
   - 경찰청 (국번없이 182)
   - 금융감독원 (1332)
   - 개인정보침해 신고센터 (국번없이 118)
```

### Q4: "개인정보 처리방침"이 너무 길어서 안 읽어요
**A:** 핵심만 확인하세요:
```
필수 확인 사항:
1. 수집 항목:
   "뭘 수집하나?"

2. 이용 목적:
   "왜 수집하나?"

3. 보유 기간:
   "언제까지 보관하나?"

4. 제3자 제공:
   "다른 회사에 주나?"

5. 파기 방법:
   "어떻게 삭제하나?"

→ 이 5가지만이라도 읽기!
```

### Q5: 개인정보를 지키려면 어떻게 해야 하나요?
**A:** 일상 속 실천:
```
🔐 온라인:
- 강력한 비밀번호 사용
- 사이트마다 다른 비밀번호
- 2단계 인증 설정
- 공용 PC에서 로그아웃
