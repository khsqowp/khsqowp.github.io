### 📚 어제 학습 내용 복습

본격적인 학습에 앞서, 어제 배운 Pandas의 기본 데이터 구조를 간단히 복습했다.

*   **`Series`**: 1차원 배열과 같은 구조로, **`인덱스(index)`**와 **`값(value)`**으로 구성된다.
*   **`DataFrame`**: 2차원 테이블 구조로, 여러 개의 `Series`가 모여 하나의 `DataFrame`을 이룬다고 볼 수 있다. 각 `Series`는 `DataFrame`의 **열(column)**이 된다.
*   **기본 인덱싱**: `DataFrame`에서 대괄호 `[]`를 사용한 기본 인덱싱은 **열 기준**으로 동작한다. 행을 선택하기 위해서는 슬라이싱(`:`)이나 전용 인덱서를 사용해야 한다.

> **💡 중요!:** `DataFrame`을 다룰 때는 개별 열을 하나의 `Series`로, 그리고 `DataFrame` 전체를 `Series`들의 집합으로 이해하는 것이 매우 중요하다. 이 관점은 데이터를 조작하고 분석하는 모든 과정의 기초가 된다.

###  podstawowe DataFrame 조작하기

본격적으로 `DataFrame`을 만들고 조작하는 방법에 대해 배웠다.

#### 딕셔너리로 DataFrame 생성하기

가장 기본적인 방법은 파이썬의 `딕셔너리`를 `DataFrame`으로 변환하는 것이다.

*   딕셔너리의 **키(key)**는 `DataFrame`의 **열 이름(column name)**이 된다.
*   딕셔너리의 **값(value)**은 각 열의 데이터가 되며, 보통 리스트 형태로 제공한다.

```python
import pandas as pd
import numpy as np

# 학생 점수 데이터가 담긴 딕셔너리
scores = {
    'kor' : [90, 85, 100, 88, 78],
    'eng' : [90, 85, 100, 88, 78],
    'mat' : [90, 85, 100, 80, 78]
}

# 딕셔너리와 인덱스를 지정하여 DataFrame 생성
frm = pd.DataFrame(scores,
                    index = ['강승우', '최호준', '임종섭', '이현우', '오신원'])

print("--- DataFrame 정보 ---")
print(frm)
```

**출력 예제:**
```
--- DataFrame 정보 ---
     kor  eng  mat
강승우   90   90   90
최호준   85   85   85
임정섭  100  100  100
이현우   88   88   88
오신원   78   78   78
```

#### 새로운 열 추가 및 계산

`DataFrame`에 새로운 열을 추가하는 것은 매우 간단하다. 기존에 없는 열 이름을 인덱스로 지정하고 값을 할당하면 된다.

> **📌 노트:** `DataFrame`의 데이터는 내부적으로 `NumPy` 배열(`ndarray`)로 관리된다. 따라서 `NumPy`에서 사용했던 축(axis) 개념과 연산 함수들을 `DataFrame`에도 거의 그대로 적용할 수 있다.

**예제: 모든 학생의 과목 평균 점수를 'mean' 열로 추가하기**

```python
# axis=1은 행(row) 방향으로 연산을 수행하라는 의미
# 즉, 각 학생(행)의 국어, 영어, 수학 점수의 평균을 계산한다.
mean_scores = frm.mean(axis=1)

# 'mean'이라는 새로운 열에 계산된 평균 점수 Series를 할당
frm['mean'] = mean_scores

print("--- 'mean' 열 추가 후 ---")
print(frm)
```

**출력 예제:**
```
--- 'mean' 열 추가 후 ---
     kor  eng  mat        mean
강승우   90   90   90   90.000000
최호준   85   85   85   85.000000
임정섭  100  100  100  100.000000
이현우   88   88   88   85.333333
오신원   78   78   78   78.000000
```

#### 특정 데이터 수정하기

`DataFrame`의 특정 위치의 값을 수정할 때는 `.loc` 인덱서를 사용하는 것이 가장 명확하고 안전한 방법이다.

> **💡 중요!:** `.loc[행_이름, 열_이름]` 형식으로 접근하면 특정 셀을 정확히 지정할 수 있다.

**예제: '최호준' 학생의 영어 점수를 90점으로 수정하고, 평균 점수 다시 계산하기**

```python
# 1. '최호준' 학생의 'eng' 점수를 90으로 수정
frm.loc['최호준', 'eng'] = 90

# 2. 수정한 점수를 바탕으로 '최호준' 학생의 평균을 다시 계산하여 'mean' 열 업데이트
#    loc를 이용해 kor, eng, mat 점수만 선택 후 평균 계산
frm.loc['최호준', 'mean'] = frm.loc['최호준', ['kor', 'eng', 'mat']].mean()

print("--- 최호준 학생 점수 수정 후 ---")
print(frm)
```

**출력 예제:**
```
--- 최호준 학생 점수 수정 후 ---
     kor  eng  mat        mean
강승우   90   90   90   90.000000
최호준   85   90   85   86.666667
임정섭  100  100  100  100.000000
이현우   88   88   80   85.333333
오신원   78   78   78   78.000000
```

### 🔍 DataFrame 인덱싱과 슬라이싱 심화

데이터를 원하는 대로 추출하려면 인덱싱 방법을 정확히 이해해야 한다.

#### `Series` vs `DataFrame` 반환

-   **하나의 열/행 선택**: `frm['kor']` 또는 `frm.loc['최호준']`과 같이 하나의 대상을 선택하면 `Series` 객체가 반환된다.
-   **여러 열/행 선택**: `frm[['kor', 'eng']]` 또는 `frm.loc[['최호준', '이현우']]`와 같이 **리스트 형태(이중 대괄호 `[[]]`)**로 여러 대상을 선택하면 `DataFrame` 객체가 반환된다.

**개념 증명 코드:**
```python
# 하나의 행을 선택 -> Series
lee_series = frm.loc['이현우']
print("--- 단일 행 선택 (Series) ---")
print(type(lee_series))
print(lee_series)

print("\n" + "="*30 + "\n")

# 리스트를 사용해 하나의 행을 선택 -> DataFrame
lee_df = frm.loc[['이현우']]
print("--- 리스트로 단일 행 선택 (DataFrame) ---")
print(type(lee_df))
print(lee_df)
```

**출력 예제:**
```
--- 단일 행 선택 (Series) ---
<class 'pandas.core.series.Series'>
kor     88.000000
eng     88.000000
mat     80.000000
mean    85.333333
Name: 이현우, dtype: float64

==============================

--- 리스트로 단일 행 선택 (DataFrame) ---
<class 'pandas.core.frame.DataFrame'>
     kor  eng  mat       mean
이현우   88   88   80  85.333333
```

#### `.loc`을 이용한 자유로운 데이터 추출

`.loc` 인덱서는 `라벨(이름)`을 기반으로 행과 열을 동시에 선택할 수 있어 매우 강력하다.

-   `frm.loc['행이름']`: 특정 행의 모든 열
-   `frm.loc[['행1', '행2']]`: 특정 행들의 모든 열
-   `frm.loc[:, '열이름']`: 모든 행의 특정 열 (`:`는 전체를 의미)
-   `frm.loc['행이름', '열이름']`: 특정 셀 하나
-   `frm.loc[['행1', '행2'], ['열1', '열2']]`: 특정 행/열들의 부분 집합

**예제: `.loc`으로 특정 범위의 데이터 가져오기**
```python
# '최호준'부터 '이현우'까지의 학생들의 'kor'부터 'eng'까지의 점수
partial_data = frm.loc['최호준':'이현우', 'kor':'eng']
print(partial_data)
```

**출력 예제:**
```
     kor  eng
최호준   85   90
임정섭  100  100
이현우   88   88
```

### ✂️ 데이터 필터링과 정렬

#### Boolean 인덱싱으로 조건에 맞는 데이터 추출하기

`DataFrame`에 특정 조건을 적용하면, 각 행이 조건에 부합하는지 여부를 `True`/`False`로 담은 `Series`가 반환된다. 이 `boolean Series`를 다시 `DataFrame`의 인덱서에 넣으면 `True`인 행들만 추출할 수 있다. 이것이 **Boolean 인덱싱**의 원리다.

**예제: 3등실 승객 데이터만 추출하기**
```python
# 'pclass'가 3인 조건 생성
condition = titanic_df['pclass'] == 3

# 조건을 만족하는(True) 행만 추출하여 새로운 DataFrame 생성
pclass_3_df = titanic_df[condition]

print("--- 3등실 승객 데이터 (상위 5개) ---")
print(pclass_3_df.head())
```
여러 조건을 조합할 때는 각 조건을 소괄호 `()`로 묶고, 논리 연산자를 사용한다.

> **💡 중요!:** Pandas에서 여러 조건을 조합할 때는 파이썬의 `and`, `or` 키워드 대신 비트 연산자인 `&` (AND), `|` (OR)를 사용해야 한다. 각 조건은 반드시 괄호로 감싸주어야 연산자 우선순위 문제를 피할 수 있다.

**예제: 1등실에 탑승한 60세 이상 여성 승객 찾기**
```python
# 여러 조건을 & 연산자로 조합
complex_condition = (titanic_df['age'] >= 60) & (titanic_df['pclass'] == 1) & (titanic_df['sex'] == 'female')

# 최종 조건에 맞는 데이터 추출
senior_female_df = titanic_df[complex_condition]

print("--- 1등실 60세 이상 여성 승객 ---")
print(senior_female_df)
```

#### 데이터 정렬하기: `sort_values()`

#### 인덱스 재설정 및 관리: `reset_index()`, `set_index()`

-   **`.reset_index()`**: 기존 인덱스를 버리고, 0부터 시작하는 새로운 정수 인덱스를 생성한다.
    > **📌 노트:** `.reset_index()`를 실행하면 기존 인덱스는 'index'라는 이름의 새로운 **열**로 `DataFrame`에 추가된다. 이것이 불필요하다면 `.drop('index', axis=1)`을 사용해 삭제하거나, `reset_index(drop=True)` 옵션을 사용하면 된다.

### 📊 그룹화와 집계 (Group By)

오늘 배운 내용의 하이라이트. `groupby`는 데이터 분석의 핵심 중의 핵심이다.

> **💡 중요!:** `.groupby()`는 데이터를 특정 기준으로 여러 그룹으로 **분할(Split)**하고, 각 그룹에 독립적으로 함수를 **적용(Apply)**한 뒤, 그 결과들을 다시 하나로 **결합(Combine)**하는 **Split-Apply-Combine** 전략을 따른다.

#### 기본 그룹화 및 집계

**예제: `tips` 데이터셋에서 성별에 따른 평균 팁 금액 비교**
```python
# 'tips' 데이터셋 로드
tips_df = sns.load_dataset('tips')

# 1. 'sex' 열을 기준으로 그룹화
# 2. 각 그룹에서 'tip' 열만 선택
# 3. 각 그룹의 평균('mean')을 계산
gender_tip_mean = tips_df.groupby('sex')['tip'].mean()

print("--- 성별 평균 팁 ---")
print(gender_tip_mean)
```

**출력 예제:**
```
--- 성별 평균 팁 ---
sex
Male      3.089618
Female    2.833448
Name: tip, dtype: float64
```
> **📌 노트:** 위 결과를 보면 남성 손님들이 여성 손님들보다 평균적으로 팁을 조금 더 많이 주는 경향이 있음을 알 수 있다. 이것이 바로 `groupby`를 통해 얻을 수 있는 **인사이트(insight)**다.

#### 여러 집계 함수 적용: `.agg()`

하나의 그룹에 대해 평균, 합계, 최대값 등 여러 통계량을 한 번에 보고 싶을 때는 `.agg()` 함수를 사용한다.

**예제: 성별에 따른 나이의 최대, 최소, 평균, 중간값 구하기**
```python
# 'gender'로 그룹화 후 'age' 열 선택
# .agg()에 원하는 통계 함수 이름들을 리스트로 전달
age_stats_by_gender = frm.groupby('gender')['age'].agg(['max', 'min', 'mean', 'median'])

print(age_stats_by_gender)
```

**출력 예제:**
```

gender

여자       40   21  29.000000    28.0
```

#### 실전 예제: `groupby`와 `apply`로 결측값 채우기

`groupby`의 가장 강력한 활용 사례 중 하나는 그룹별 특성을 이용해 데이터를 전처리하는 것이다. `titanic` 데이터의 `age` 열에는 결측값이 많았다. 이 결측값을 전체 나이 평균으로 채우는 것보다, **각 성별 그룹의 평균 나이**로 채우는 것이 훨씬 더 정교한 방법이다.

**1. 성별 평균 나이 확인**
```python
print(titanic_df.groupby('sex')['age'].mean())
# sex
# female    27.915709
# male      30.726645
# Name: age, dtype: float64
```

**2. `apply`와 `lambda`를 이용해 결측값 채우기**
```python
# 'age' 열의 결측값 개수 확인
print('수정 전 age 결측값 개수:', titanic_df['age'].isnull().sum())

# 1. 'sex'로 그룹화하고 'age' 열 선택
# 2. 각 그룹(x)에 대해 apply 함수 적용
# 3. lambda 함수는 각 그룹(Series)을 받아, 그 그룹의 결측값을 그룹의 평균(x.mean())으로 채움(fillna)
filled_age_series = titanic_df.groupby('sex')['age'].apply(lambda x: x.fillna(x.mean()))

# 4. 원본 DataFrame의 'age' 열을 새로 생성된 Series로 교체
titanic_df['age'] = filled_age_series

print('수정 후 age 결측값 개수:', titanic_df['age'].isnull().sum())
```

**출력 예제:**
```
수정 전 age 결측값 개수: 177
수정 후 age 결측값 개수: 0
```
> **💡 중요!:** 위 코드는 단 두 줄로 복잡한 조건부 결측값 대체를 완료했다. `for` 루프와 `if` 문을 사용했다면 훨씬 더 코드가 길고 복잡해졌을 것이다. 이처럼 `groupby`는 Pandas를 이용한 효율적인 데이터 처리의 핵심이다.

#### 🔐 보안 노트: `pd.read_csv` 사용 시 주의점

강의에서 `pd.read_csv()`를 사용하여 로컬 파일을 읽는 방법을 배웠다. 이 기능은 매우 편리하지만, 만약 파일 경로를 사용자로부터 직접 입력받는 애플리케이션을 만든다면 보안상 주의가 필요하다.

-   **잠재적 위협: 경로 조작 (Path Traversal)**
    악의적인 사용자가 `../../../../etc/passwd`와 같이 상위 디렉터리로 이동하는 경로를 입력하여 시스템의 민감한 파일에 접근을 시도할 수 있다.

-   **대응 방안: 경로 검증**
    사용자로부터 입력받은 파일 경로는 반드시 검증해야 한다. 예상된 디렉터리 내에 파일이 위치하는지 확인하고, `..` 와 같은 상위 디렉터리 이동 문자를 필터링해야 한다.

**개념 증명 코드 (안전한 경로 처리 예시):**
```python
import os

def safe_read_csv(user_input_path):
    # 1. 파일을 저장해 둔 기본 디렉터리 정의
    base_dir = os.path.abspath('./SK_Rookies/data')
    
    # 2. 사용자 입력 경로와 기본 디렉터리를 합쳐 절대 경로 생성
    requested_path = os.path.abspath(os.path.join(base_dir, user_input_path))
    
    # 3. 최종 경로가 기본 디렉터리 내에 있는지 확인
    if os.path.commonprefix([requested_path, base_dir]) != base_dir:
        # 기본 디렉터리를 벗어나는 경우 에러 발생
        raise ValueError("허용되지 않은 파일 경로입니다.")
    
    # 4. 안전한 경우에만 파일 읽기
    try:
        print(f"'{requested_path}' 파일을 안전하게 읽습니다.")
        return pd.read_csv(requested_path)
    except FileNotFoundError:
        raise FileNotFoundError("파일을 찾을 수 없습니다.")
    except Exception as e:
        raise e

# 안전한 사용 예
try:
    baby_names = safe_read_csv('year2022_baby_name.csv')
    print("파일 읽기 성공!")
except Exception as e:
    print(f"에러 발생: {e}")

# 위험한 사용 예
try:
    # 상위 디렉터리로 접근 시도
    sensitive_data = safe_read_csv('../../../../../etc/hosts') 
except Exception as e:
    print(f"에러 발생: {e}")
```

**출력 예제:**
```
'/Users/user/Movies/room/SK_Rookies/data/year2022_baby_name.csv' 파일을 안전하게 읽습니다.
파일 읽기 성공!
에러 발생: 허용되지 않은 파일 경로입니다.
```
