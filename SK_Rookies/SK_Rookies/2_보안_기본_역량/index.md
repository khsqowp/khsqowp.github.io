# 2. 보안 기본 역량

사이버 보안 전문가가 되기 위한 핵심 보안 지식과 기술을 습득합니다.

## 학습 목표
- 네트워크 보안의 이론과 실전
- 클라우드 보안 아키텍처
- 애플리케이션 보안 및 시큐어 코딩
- 개인정보보호 법규 및 실무

## 하위 모듈
- **2-1. 네트워크 보안**: 모든 사이버 보안의 기초
- **2-2. 클라우드 보안**: 현대 인프라 보안의 필수 요소
- **2-3. 애플리케이션 보안**: 소프트웨어 보안의 실전
- **2-4. 개인정보보호**: 법규 준수와 윤리적 책임
