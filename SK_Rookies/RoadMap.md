# SK Rookies 생성형 AI 사이버보안 마스터 로드맵

## Ⅰ. 과정 목표 및 개요

- **과정 목표:** 클라우드와 AI 기술을 융합하여 실전적인 위협 대응 능력을 갖춘 사이버보안 전문가를 양성합니다.
- **핵심 역량:** ①보안 기본기 ②AI 활용 능력 ③실무 프로젝트 해결 능력
- **최종 산출물:** 생성형 AI를 활용한 보안 문제 해결 프로젝트 포트폴리오

#### **Part 2. 보안 기본 역량 (Security Fundamentals)**

##### **1-3. 네트워크 보안 (56H)**
- **목표:** 모든 보안의 기초가 되는 네트워크의 구조와 통신 원리를 이해하고, 주요 공격 및 방어 기법을 학습합니다.
- **사전 지식:**
  - IT 인프라에 대한 기본적인 관심
- **사용 도구:**
  - `Wireshark`: 네트워크 패킷 캡처 및 상세 분석
  - `Nmap`: 포트 스캐닝 및 네트워크 정보 수집
- **핵심 학습 내용:**
  - **네트워크 기초:** OSI 7계층, TCP/IP 모델, IP 주소(IPv4/IPv6), 서브네팅
  - **주요 프로토콜:** HTTP/HTTPS, DNS, TCP, UDP, ICMP 등 프로토콜별 구조와 기능
  - **네트워크 공격:** DDoS, 스니핑, 스푸핑, 하이재킹 등 원리 및 실습
  - **보안 솔루션:** 방화벽, IDS/IPS, VPN의 역할과 작동 원리
- **심화 응용:**
  - `Wireshark`로 캡처한 패킷을 분석하여 비정상 통신 패턴 식별
  - 가상 환경에서 간단한 네트워크 공격 시나리오 재현

##### **1-4. 클라우드 보안 (56H)**
- **목표:** 현대 IT 환경의 중심인 클라우드(AWS)의 보안 아키텍처를 이해하고, 안전하게 인프라를 구축 및 운영하는 역량을 기릅니다.
- **사전 지식:**
  - 네트워크 기본 지식 (1-3 과정)
- **사용 플랫폼 및 도구:**
  - **플랫폼:** Amazon Web Services (AWS)
  - **핵심 서비스:** IAM, VPC, EC2, S3, Security Groups, GuardDuty
- **핵심 학습 내용:**
  - **클라우드 개념:** IaaS, PaaS, SaaS, 책임 공유 모델
  - **계정 및 접근 관리:** `IAM`을 통한 사용자, 그룹, 역할, 정책 기반의 최소 권한 원칙 적용
  - **네트워크 격리:** `VPC`, `Subnet`, `Security Group`, `NACL`을 활용한 네트워크 보안 아키텍처 설계
  - **클라우드 위협:** 설정 오류, 계정 탈취 등 클라우드 환경의 주요 보안 위협 및 대응 방안
- **심화 응용:**
  - AWS 환경에 3-Tier 아키텍처(Web-WAS-DB)를 보안 설정에 맞게 구축
  - `GuardDuty` 등 보안 서비스를 활용하여 가상 위협 탐지 및 분석

##### **1-5. 애플리케이션 보안 (56H)**
- **목표:** 웹/모바일 서비스의 취약점을 진단하고, 개발 단계부터 보안을 고려하는 시큐어 코딩 역량을 함양합니다.
- **사전 지식:**
  - 웹 기초 (HTML, JavaScript), Python 등 프로그래밍 언어 1개 이상
- **사용 도구:**
  - `Burp Suite`, `OWASP ZAP`: 웹 프록시를 이용한 취약점 분석
  - `SQLMap`: SQL 인젝션 자동화 공격 도구
- **핵심 학습 내용:**
  - **웹 취약점:** OWASP TOP 10 (SQL Injection, XSS, CSRF 등) 원리, 공격 시나리오, 대응 방안
  - **시큐어 코딩:** 입력값 검증, 에러 처리, 세션 관리 등 개발 단계별 보안 강화 코드 작성법
  - **보안 분석:** 정적 분석(SAST), 동적 분석(DAST)의 개념과 도구 활용법
- **심화 응용:**
  - 취약하게 구축된 웹 애플리케이션(ex: DVWA) 모의해킹
  - 코드 리뷰를 통해 소스코드 상의 잠재적 보안 약점 식별 및 수정 제안

##### **1-6. 개인정보보호 (16H)**
- **목표:** 개인정보보호 관련 법규를 이해하고, 기술적/관리적 보호조치를 적용할 수 있는 역량을 갖춥니다.
- **핵심 학습 내용:**
  - **관련 법규:** 개인정보보호법, 정보통신망법의 주요 조항
  - **보호 조치:** 개인정보의 생명주기(수집-이용-제공-파기) 관리, 암호화, 접근 통제 등
  - **컴플라이언스:** 개인정보 영향평가(PIA), 개인정보 처리 방침 수립의 이해
- **심화 응용:**
  - 기업의 개인정보 처리 방침을 분석하여 법적 요구사항 준수 여부 검토

### **2단계: 응용 기술 심화 (Application)**

#### **2-1. 생성형 AI 활용 AI 보안관제 및 통합로그 분석 (64H)**
- **목표:** 생성형 AI를 활용하여 방대한 보안 로그를 자동으로 분석하고, 위협 인텔리전스를 생성하는 실무 능력을 배양합니다.
- **사전 지식:** 1단계 전 과정
- **사용 기술 및 도구:**
  - **로그 분석:** `Elastic Stack(ELK)`, `Splunk`
  - **AI 기술:** 자연어 처리(NLP), 이상 탐지(Anomaly Detection) 알고리즘, LLM API
- **핵심 학습 내용:**
  - **보안 관제:** SIEM 시스템의 이해, 위협 탐지 Rule 작성 및 이벤트 분석
  - **로그 전처리:** 정규표현식을 이용한 비정형 로그 데이터의 정형화
  - **AI 기반 분석:** 머신러닝으로 비정상 로그 패턴 학습 및 탐지, LLM으로 로그 내용 요약 및 자연어 질의
- **심화 응용:**
  - 생성형 AI를 이용해 최신 공격 기법에 대한 탐지 Rule 초안 자동 생성
  - 자연어 질문(ex: "어제 발생한 로그인 실패 이벤트 요약해줘")에 답하는 로그 분석 챗봇 프로토타입 개발

#### **2-2. 생성형 AI 활용 악성코드 분석 및 대응 (64H)**
- **목표:** AI를 활용하여 악성코드의 정적/동적 분석을 효율화하고, 자동화된 대응 체계를 구축하는 능력을 기릅니다.
- **사전 지식:** C/C++, 어셈블리어, OS 구조(PE/ELF 포맷, 프로세스, 메모리)
- **사용 도구:**
  - **분석 도구:** `IDA Pro`, `x64dbg`, `Ghidra`, `Cuckoo Sandbox`
  - **AI 기술:** `Yara`, 머신러닝 기반 악성코드 분류, LLM API
- **핵심 학습 내용:**
  - **악성코드 분석:** 정적 분석(코드 구조, 문자열), 동적 분석(행위, 네트워크 통신)
  - **리버스 엔지니어링:** 디스어셈블리, 디버깅을 통한 악성코드 동작 원리 역추적
  - **AI 기반 분류:** 파일 특징을 추출하여 머신러닝으로 악성/정상 분류 모델 생성
- **심화 응용:**
  - 생성형 AI로 난독화된 악성코드 스크립트를 해석하고, 핵심 기능 요약
  - 분석된 악성코드의 핵심 행위를 기반으로 `Yara` Rule 자동 생성

#### **2-3. 생성형 AI 활용 취약점 진단 및 모의해킹 (64H)**
- **목표:** 생성형 AI를 도구로 활용하여 모의해킹의 효율성을 극대화하고, 발견된 취약점에 대한 종합적인 보고서 작성을 자동화합니다.
- **사전 지식:** 1, 2단계 전 과정
- **사용 도구:**
  - **공격 프레임워크:** `Metasploit`
  - **AI 기술:** LLM API, 코드 생성 모델
- **핵심 학습 내용:**
  - **모의해킹 방법론:** 정보 수집 -> 취약점 분석 -> 공격 수행 -> 보고서 작성 전 과정
  - **AI 활용 공격:** 대상 시스템 환경에 맞는 공격 코드(Exploit) 초안 생성
  - **AI 활용 방어:** 발견된 취약점에 대한 보안 패치 코드 초안 생성
  - **보고서 자동화:** 진단 결과와 조치 방안을 포함한 모의해킹 결과 보고서 자동 생성
- **심화 응용:**
  - 특정 CVE(공개된 취약점) 번호에 대한 공격 개념증명(PoC) 코드를 LLM을 이용해 생성 및 분석
  - 모의해킹 시나리오 전체를 LLM과 협력하여 수행 (정보수집 -> 공격 -> 보고)

#### **3-1. 최종 실무 프로젝트 및 멘토링 (328H)**
- **목표:** 1, 2단계에서 배운 모든 지식과 기술을 총동원하여, 실제 기업이 겪을 법한 보안 문제를 정의하고 해결하는 완전한 프로젝트를 수행합니다.

  - AI를 활용한 클라우드 인프라 보안 설정 오류 자동 진단 및 수정 도구 개발

  - **고도화 (48H):** SK쉴더스 현직자 멘토링을 통한 기술적 난관 해결 및 방향성 재설정
  - **발표 (8H):** 최종 결과물 발표, 시연, 질의응답 (프로젝트 발표회)
- **필수 역량:**
  - **기술:** Python, AI/ML, Cloud, Security 전반
  - **협업:** `Git`, `Github`를 통한 버전 관리 및 협업
  - **소프트스킬:** 문제 해결 능력, 커뮤니케이션, 문서 작성, 프레젠테이션 스킬
